# tic-tac-toe

Tic Tac Toe game using HTML, CSS, and JavaScript.

Allows for two players to take turns
Detects when the game ends (when there is no more possible moves, i.e. all squares have either an X or an O). 
Detects when a player wins. To win, the player must get three of their pieces (X or O) in a row.
Outputs message to indicate whether its player X's turn or player O's turn. 
Maintains the score for each player, i.e. the number of games that player has won. Reset button resets the scores as well as the board.
